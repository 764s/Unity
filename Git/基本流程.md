### [克隆仓库到本地](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository)  
1. Open Git Bash  
2. 进入文件夹  
3. 输入`git clone`  
```
// 会自带外壳文件夹
$ git clone https://github.com/YOUR-USERNAME/YOUR-REPOSITORY
```
